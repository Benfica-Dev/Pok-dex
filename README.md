# Pokedéx 

Projeto de Pokedéx, inspirada no jogo Pokémon GO e no anime e mangá Pokémon. Desenvolvido em JavaScript, com foco em manipulação do DOM, armazenamento de informações e lógica do código. O usuário inicia com um espaço vazio e sorteia pókemons por meio de um sistema de gacha. Os dados são salvos no navegador para que não sejam perdidos.

## Funcionalidades

- Armazenar pokémons
- Identificá-los por categorias
- Guardar dados
- Sortear pókemons aleatoriamente

## Limitações

- Não possui banco de dados

## Tecnologias Utilizadas 

- HTML%
- CSS3
- JavaScript (Vanilla JS)
- PokéAPI
